During my second year of university, my team and I had to create an e-commerce website containing computer peripherals. In this project, my technical role was to work as a front-end developer.
I learnt how to use the Tailwind CSS framework, as well as using Git and Github for the first time with a team. We also utilised the Laravel PHP framework, using Laravel Blade to build our pages. This project taught me a lot about web development and significantly improved my coding skills.
